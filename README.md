
# AquaTask

AquaTask is a sleek, modern, and fully responsive **To-Do List** web application designed to boost your productivity. Built with a clean and attractive UI, it works flawlessly on all devices — desktops, tablets, and mobiles.

## Features

- **Responsive Design**  

  Fully optimized for all screen sizes, ensuring a seamless experience on mobile and desktop.

- **User-Friendly Interface**  

  Intuitive and elegant design with smooth interactions to help you manage your tasks effortlessly.

- **Task Management**  

  Add, edit, delete, and mark tasks as completed. Data is stored persistently in a lightweight SQLite database.

- **Open Source**  

  Free to use, modify, and distribute. Contributions and feedback are welcome!
